# catversions

Silly CLI utility for checking project versions

## Usage
Actual folder tree:
```
.
├── project1
│   ├── src
│   └── version
├── project2
│   ├── lib
│   ├── src
│   └── version
└── project3
    ├── tests
    └── version
```

The output will be:
```
project1 1.0.0
project2 1.2.5
project3 2.0.1.2.dev
```
